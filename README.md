# Docker Graylog

This repository demonstates configuring Graylog to centralize log management on a network.

The guide specifically will demonstrate installation on an Ubuntu 20.04 Server hosted on ProxMox and uses Systemd & Docker to manage the log service.

However, this guide can be used generally for any Linux server that supports Docker.

## Repository Organization

`assets` directory: contains any images used in the documentation
`docs` directory: contains the documentation for using the source files in this repository
`single-node` directory: contains the source files for provisioning a single graylog node
